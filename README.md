## ▶️ How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/SURAJJJJJ11111/SummerPEP_Day4.git
   ```

2. Open the desired project in **Spring Tool Suite (STS)** or **IntelliJ IDEA**.

3. Update Maven dependencies.

4. Run the project as a Spring Boot Application.

5. Access Swagger UI (if available):

   ```
   http://localhost:8080/swagger-ui/index.html
   ```
